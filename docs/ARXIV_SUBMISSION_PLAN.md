## Files to prepare

arXiv prefers TeX/LaTeX source when available. Prepare:

```text
paper/main.tex
figures/*.png or *.pdf
bibliography if separated
```

Also keep the GitHub repository as an external verification package, but do not rely on GitHub links as the only record.

## Endorsement note

A new arXiv author or a new submission category may require endorsement. Do not mass-email endorsers. Send a brief, respectful request only to an appropriate eligible endorser with the manuscript and repository package.
